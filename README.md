TapBuy is a web-based e-commerce application developed using Java (JSP & Servlets). The system allows users to browse products while administrators can manage product listings through an admin panel.

Features
- User product browsing
- Admin login and dashboard
- Add, update, and manage products
- Responsive UI using Bootstrap

Technologies Used
- Java (JSP, Servlets)
- HTML, CSS, JavaScript
- Bootstrap
- Hibernate ORM
- MySQL Database
- GlassFish Server
- NetBeans IDE

Project Architecture
- MVC (Model-View-Controller)
- JSP for views
- Servlets for business logic
- Hibernate for database persistence
- MySQL for data storage

How to Run
1. Clone the repository  
2. Open the project in NetBeans  
3. Configure MySQL database and Hibernate (`hibernate.cfg.xml`)  
4. Deploy the application on GlassFish Server  
5. Run the application from the server

Purpose
This project was developed for learning Java EE web development concepts, Hibernate ORM, database persistence, and MVC architecture.
